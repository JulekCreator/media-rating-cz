# Browser Extension - Instalace

## Jak nainstalovat rozšíření do Chrome/Edge/Bing

### Krok 1: Stáhněte soubory
1. Stáhněte celý repozitář jako ZIP:
   - Klikněte na zelené tlačítko "Code" nahoře
   - Vyberte "Download ZIP"
2. Rozbalte ZIP soubor na svém počítači
3. Najděte složku `extension` uvnitř

### Krok 2: Otevřete nastavení rozšíření
**Chrome:**
1. Otevřete Chrome
2. Přejděte na `chrome://extensions/`
3. Zapněte "Vývojářský režim" (Developer mode) v pravém horním rohu

**Edge:**
1. Otevřete Edge
2. Přejděte na `edge://extensions/`
3. Zapněte "Vývojářský režim" v levém dolním rohu

### Krok 3: Načtěte rozšíření
1. Klikněte na "Načíst rozšíření bez balíčku" (Load unpacked)
2. Vyberte složku `extension` ze stažených souborů
3. Rozšíření by se mělo objev it v seznamu

### Krok 4: Otestujte rozšíření
1. Otevřete některý český zpravodajský server (např. idnes.cz, aktualne.cz)
2. Nahoře na stránce by se měl objevit barevný panel s hodnocením média

## Funkce

- **Automatické rozpoznání** - Rozšíření automaticky detekuje, které médium navštěvujete
- **Kompaktní panel** - Horní lišta s přehledným hodnocením
- **Detailní metriky** - Přesnost, Transparentnost, Profesionalita, Vyváženost
- **Zavíratel né** - Panel lze zavřít kliknutím na křížek
- **Responzivní** - Funguje na počítačich i mobilech

## Poznámky

- Hodnocení jsou **demonstrační** a slouží jako ukázka konceptu
- Pro produkční nasazení je třeba implementovat reálná data a metodologii
- Rozšíření nesbrá žádná data o uživateli
